Project Description

Game engine is very useful thing for game programmers. A simple game engine is just a library with game programming related classes or functions. How I become game engine developer? OK, I explain it.

I bought RZX-50 because I like emulation and old-school video games. RZX-50 similar to Dingoo A320, but RZX-50 use Dingux as firmware by default. Dingux is Linux bases operating system. I made my own multimedia library for Dingux because I want test myself as Linux programmer. This library has a cross-platform layered architecture. It has a two kinds of subsystems: system-specific and common subsystems. Common subsystems working over system-specif subsystems.

I also made few multimedia libraries for Microsoft Windows. I done it by easy way: just forked my Dingux library and replace all system-specific subsystems.
