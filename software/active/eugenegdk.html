Project Description

Eugene game development kit is ultra-fast 2D game engine for Windows 2000 or higher. It uses OPENGL for render.

Features

A good game engine should have optimal set of features. Let's look to features of my game engine.

  • Easy to use object-orientated API.
  • Graphics primitives.
  • Animated and static sprites.
  • Scrollable and static backgrounds.
  • Tile-maps.
  • Image transformation subsystem: resizing and mirroring.
  • Image loader with TGA support.
  • Text drawing subsystem.
  • Custom fonts.
  • Timers.
  • Collision detector.
  • Integrated fps counter.
  • Powerful input subsystem: keyboard, mouse, game-pad.
  • Simple multimedia subsystem for playing audio and video.
